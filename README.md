# 🍽️ 今天吃什么

微信小程序 — 帮你决定每一餐

## 功能

### 🎰 随机转盘
- Canvas 2D 转盘旋转动画，支持分类筛选
- 老虎机随机推荐（文字轮播 + 减速效果）
- 自定义食物：添加、编辑、删除，本地持久化
- 63 个预设食物（中餐/西餐/日料/韩餐/东南亚/快餐）

### 🧊 冰箱推荐（静态匹配 + AI fallback）
- 选择冰箱里的食材，从预置菜谱库中按食材匹配率排序推荐
- 匹配不足时自动 fallback 到 AI 生成（讯飞星火 API）
- 菜谱详情：用料清单、步骤预览（展开/收过）
- 匹配度标识：显示匹配率百分比
- 缺少食材 → 一键生成购物清单，可复制

### ❤️ 收藏功能
- 推荐的菜谱可一键收藏（❤️ / 🤍）
- 「我的」页面展示收藏列表
- 收藏菜谱可查看完整用料和步骤
- 本地存储，重启不丢失

### 📖 菜谱浏览
- **47 道预置菜谱**（中餐 19 / 西餐 7 / 日料 6 / 韩餐 5 / 减脂 10）
- 菜谱列表：分类筛选、搜索，秒开无等待
- 菜谱详情：用料、步骤、营养信息
- 减脂餐专区：热量筛选、营养成分展示
- 规则引擎打分推荐：分类匹配 + 食材匹配 + 关键词 + 难度 + 随机因子

### 🗳️ 多人投票
- 创建投票房间（6 位房间码）
- 加入房间、提交投票、查看结果

### 👤 个人中心
- 口味偏好（辣度/甜度/酸度）
- 饮食目标（减脂/增肌/均衡）
- 过敏源和忌口设置
- 我的收藏（展开/收起/删除）

## 推荐架构

```
┌─────────────────────────────────────────────────┐
│              菜谱推荐引擎 (recipe-engine.js)       │
├─────────────────────────────────────────────────┤
│  第一层：静态数据 (data/recipes.js)               │
│  → 47 道预置菜谱，require 即用，零延迟           │
├─────────────────────────────────────────────────┤
│  第二层：规则引擎 (recommend / matchByIngredients)│
│  → 打分排序：分类+食材+关键词+难度+随机因子       │
├─────────────────────────────────────────────────┤
│  第三层：AI fallback（仅冰箱推荐）                 │
│  → 静态匹配不足时，调用讯飞星火 API 生成          │
└─────────────────────────────────────────────────┘
```

## 技术栈

- **前端**：微信小程序原生（WXML + WXSS + JS）
- **推荐引擎**：静态 JSON + 规则打分算法
- **AI（可选）**：讯飞星火 Anthropic 兼容 API（仅冰箱推荐 fallback）
- **图标**：手绘风格 PNG 图标（34 个：20 食材 + 6 分类 + 8 菜品）
