# 🍽️ 今天吃什么

微信小程序 — 帮你决定每一餐

## 功能

### 🎰 随机转盘
- Canvas 2D 转盘旋转动画，支持分类筛选
- 老虎机随机推荐（文字轮播 + 减速效果）
- 自定义食物：添加、编辑、删除，本地持久化
- 63 个预设食物（中餐/西餐/日料/韩餐/东南亚/快餐）

### 🧊 冰箱推荐（AI 驱动）
- 选择冰箱里的食材，AI 智能推荐菜谱
- 讯飞星火 API 直接调用，无需云函数部署
- 菜谱详情：用料清单、步骤预览（展开/收过）
- 匹配度标识：✅ 能做 / ⭐ 只差一步
- 缺少食材 → 一键生成购物清单，可复制

### ❤️ 收藏功能
- AI 推荐的菜谱可一键收藏（❤️ / 🤍）
- 「我的」页面展示收藏列表
- 收藏菜谱可查看完整用料和步骤
- 本地存储，重启不丢失

### 📖 菜谱浏览
- 菜谱列表：分类筛选、搜索、排序（热门/快手/低卡）
- 菜谱详情：用料、步骤、营养信息、视频教程
- 减脂餐专区：热量筛选、营养成分展示

### 🗳️ 多人投票
- 创建投票房间（6 位房间码）
- 加入房间、提交投票、查看结果

### 👤 个人中心
- 口味偏好（辣度/甜度/酸度）
- 饮食目标（减脂/增肌/均衡）
- 过敏源和忌口设置
- 我的收藏（展开/收起/删除）

## 技术栈

- **前端**：微信小程序原生（WXML + WXSS + JS）
- **后端**：微信云开发（云函数 + 云数据库）
- **AI**：讯飞星火 Anthropic 兼容 API
- **图标**：手绘风格 PNG 图标（34 个：20 食材 + 6 分类 + 8 菜品）
